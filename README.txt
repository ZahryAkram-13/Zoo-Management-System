Zoo Management System

Overview:
This Zoo Management System is a web application designed to efficiently manage information about zoo animals.
The project follows the Model-View-Controller (MVC) architectural pattern for a structured and organized codebase.

Features:
Create, Read, Update, Delete (CRUD): Utilizes CRUD operations to seamlessly manipulate animal data.
MVC Architecture: Implements a modular structure for easy maintenance and scalability.
User-Friendly Interface: Provides an intuitive interface for interacting with zoo animal records.
JSON Integration: Supports RESTful principles, offering JSON data representations for seamless integration with other systems.

Usage:
Navigate to the homepage to explore all available animals in the zoo.
Click on an animal's name to view detailed information.
Perform CRUD operations via the provided forms for adding, updating, or deleting animals.
